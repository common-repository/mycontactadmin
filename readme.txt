== Description ==

[MyContactAdmin](http://www.olsys-mediaweb.fr/plugin_wordpress) allows you to add a contact insert on your WordPress dashboard.

Thanks to the visualization interface, you see live the text indicated in the fields of the form.

You can change the fields at any time and view the new content.

All form changes can be changed in the "Settings" tab when editing forms in the WordPress administration area.


For more information, check out plugin page at [MyContactAdmin](http://www.olsys-mediaweb.fr/plugin_wordpress).

= Features include: =

* Adding a custom insert
* Possibility of personalizing the text
* Identification of your company
* Identification of the contact to be contacted
* Customizing your opening hours

= Get involved =

Feel free to contribute to the source code via the contact section.

== Installation ==

1. Install MyContactAdmin either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the MyContactAdmin settings and set your options.
